# go-crud
My public Restful API with Golang

# Prerequisition
- Install Go compiler to your computer
- Pull code to your `$GOPATH/src`
- Install Glide package manager for Golang

# Glide Install
```
brew install glide
```

# Installation
```
glide install
```

# Application routes
## GET /users
```
curl http://localhost:3000/users
```
